# SEO 规则与 Slug 锁定指南

为了确保网站 URL 的稳定性和 SEO 权重的持续积累，特制定以下规则：

## 1. rootnotouch 锁定原则
- **定义**：`rootnotouch` 是 Markdown 文件中用于标识页面原始路径的核心字段。
- **锁定规则**：一旦 `rootnotouch` 字段生成并提交到仓库，**严禁使用 AI 自动化脚本或工具对其进行修改**。
- **手动修改唯一性**：任何对 `rootnotouch` 的变更必须由人类开发者手动编辑并提交。

## 2. AI 职责边界
- **允许操作**：AI 可以根据 `rootnotouch` 优化用于多语言展示的 `slug` 字段。
- **禁止操作**：AI 不得触碰、重写或删除 `rootnotouch` 字段。

## 3. 技术保障
- 仓库中已部署 `pre-commit` 钩子，用于在提交前检测 `rootnotouch` 的非授权变动。
- 任何违反此规则的自动化操作将被 Git 拦截。
