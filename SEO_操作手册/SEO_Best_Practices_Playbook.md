# Sinopeakchem SEO Content Strategy Playbook (2026 Edition)

Based on the analysis of high-performing "Borax" series articles, this playbook outlines the standard for creating "World-Class" B2B SEO content for industrial chemicals.

## 1. Content Architecture: The "Pillar & Cluster" Model

*   **Pillar Pages (The Authority)**: Comprehensive, 3000-4000+ word guides (e.g., *The Complete Guide to Borax*). Focus on broad, high-volume keywords.
*   **Cluster Pages (The Specialist)**: Targeted, 800-1500 word articles focusing on specific applications, comparisons, or procurement strategies (e.g., *Borax in Glass Manufacturing*, *Decahydrate vs Pentahydrate*).
*   **Internal Linking**: Every Cluster page MUST link back to the Pillar page using the primary keyword as anchor text.

## 2. On-Page SEO Standards

### A. Metadata & Frontmatter
*   **Slug**: Short, keyword-rich, and lowercase (e.g., `borax-procurement-strategies`).
*   **Title Tag**: 50-60 characters. Start with the primary keyword.
*   **Meta Description**: 150-160 characters. Must include a call to action (CTA) and the primary keyword.
*   **Excerpt**: A concise technical summary (2-3 sentences) for search snippets.

### B. Header Hierarchy (H1-H4)
*   **H1**: Matches the Title Tag exactly.
*   **H2**: Covers main topics. Use semantic variations of keywords.
*   **H3/H4**: Break down technical details, specs, or sub-applications.

### C. The "Technical Trust" Section
*   **Comparison Tables**: Always include a Markdown table for specifications (Purity, CAS, B2O3 content, etc.).
*   **Decision Matrices**: Use tables to help buyers choose between grades (e.g., "If your priority is X, choose Y").
*   **Citations**: Use a "Technical Sources & References" section at the end to build E-E-A-T (Experience, Expertise, Authoritativeness, and Trustworthiness).

## 3. B2B Copywriting Logic

*   **The "Procurement Angle"**: Don't just talk about chemistry; talk about **Total Cost of Ownership (TCO)**, **Logistics Efficiency**, and **Supply Chain Risk**.
*   **Strategic Boldness**: Use **bold text** for key terms, chemicals, and brand names (Sinopeakchem) to improve scannability.
*   **Call to Action (CTA)**: Every article must end with a clear CTA, such as "Request a Quote" or "Download Technical Data Sheet," linked to the contact page.

## 4. Visual SEO
*   **Image Alt Text**: Descriptive and keyword-rich (e.g., "High-purity Borax crystals in industrial laboratory").
*   **File Naming**: Keywords in file names (e.g., `borax-industrial-applications.webp`).

## 5. Schema Markup
*   **TechArticle**: For deep-dive guides.
*   **Product**: For specification-heavy pages.
*   **FAQ**: For common buyer questions.
