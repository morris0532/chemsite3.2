```
                    ┌─────────────────────────────────┐
                    │     PILLAR PAGE (支柱页)          │
                    │  The Ultimate Guide to           │
                    │  Sodium Bicarbonate              │
                    │  (全面指南，3000-5000字)           │
                    └──────────┬──────────────────────┘
                               │
          ┌────────────────────┼────────────────────┐
          │                    │                    │
    ┌─────┴─────┐      ┌──────┴──────┐     ┌──────┴──────┐
    │ Cluster A  │      │ Cluster B   │     │ Cluster C   │
    │ 产品与规格  │      │ 应用场景     │     │ 采购指南     │
    │ (6篇)      │      │ (8篇)       │     │ (6篇)       │
    └─────┬─────┘      └──────┬──────┘     └──────┬──────┘
          │                    │                    │
    ┌─────┴─────┐      ┌──────┴──────┐     ┌──────┴──────┐
    │ Cluster D  │      │ Cluster E   │     │ Cluster F   │
    │ 行业市场    │      │ 物流进口     │     │ 技术对比     │
    │ (4篇)      │      │ (3篇)       │     │ (3篇)       │
    └───────────┘      └─────────────┘     └─────────────┘
```

### 各 Cluster 定义

| Cluster | 主题方向 | 目标用户 | 核心关键词群 |
|---------|---------|---------|-------------|
| **A. 产品与规格** | 产品等级、规格参数、生产工艺 | 技术采购、质检人员 | specifications, food grade, USP grade, COA, production process |

| **C. 采购指南** | 如何选供应商、价格、MOQ、样品 | 采购经理、贸易商 | supplier, price, wholesale, buy, MOQ, how to choose |
| **D. 行业市场** | 市场趋势、价格走势、产能分析 | 决策者、分析师 | market size, price trend, demand forecast, industry analysis |

## 5. 顶级SEO设置规范

### 5.1 Title Tag 规范

| 规则 | 说明 |
|------|------|
| 长度 | 50-60字符（含空格） |
| 格式 | `主关键词: 副标题 \| 品牌名` 或 `主关键词: 副标题 [年份]` |
| 关键词位置 | 主关键词尽量放在Title最前面 |
| 吸引点击 | 加入数字、年份、方括号 [2026]、[Guide]、[Updated] |
| 品牌名 | 可选择在末尾加 `\| YourBrandName` |

**示例**：
```
Sodium Bicarbonate Price 2026: Per Ton Cost, Trends & Buying Tips | BrandName
```

### 5.2 Meta Description 规范

| 规则 | 说明 |
|------|------|
| 长度 | 150-160字符 |
| 内容 | 包含主关键词 + 价值主张 + CTA暗示 |
| 语气 | 专业但有吸引力 |
| 避免 | 不要堆砌关键词，不要用"Click here" |

### 5.3 URL Slug 规范

| 规则 | 示例 |
|------|------|
| 简短、含关键词 | `/blog/sodium-bicarbonate-price-guide` |
| 使用连字符分隔 | `/blog/food-grade-vs-industrial-grade-sodium-bicarbonate` |
| 不含日期 | ❌ `/blog/2026/04/sodium-bicarbonate-price` |
| 全小写 | ❌ `/blog/Sodium-Bicarbonate-Price` |

### 5.4 Heading 标签规范

```

## 7. 关键词布局策略

### 7.1 单篇文章关键词布局

| 位置 | 关键词要求 | 说明 |
|------|-----------|------|
| **Title Tag** | 主关键词 | 尽量靠前放置 |
| **Meta Description** | 主关键词 + 1个次要关键词 | 自然融入，不堆砌 |
| **URL Slug** | 主关键词 | 简短形式 |
| **H1** | 主关键词（可变体） | 与Title略有不同 |
| **首段（前100字）** | 主关键词 | 第1-2句内出现 |
| **H2标题** | 次要关键词 / LSI关键词 | 每个H2包含不同关键词 |
| **H3标题** | 长尾关键词 | 自然融入 |
| **正文** | 主关键词2-3次 + 次要关键词各1-2次 | 自然分布，不堆砌 |
| **图片Alt** | 主关键词或变体 | 描述性语言 |
| **内链锚文本** | 目标页面的关键词 | 精确匹配或部分匹配 |
| **最后一段** | 主关键词 | 总结性语句中包含 |

### 7.2 关键词密度控制

| 关键词类型 | 目标密度 | 说明 |
|-----------|---------|------|
| 主关键词 | 1.0%-1.5% | 2000字文章中出现约8-12次 |
| 次要关键词 | 0.5%-0.8% | 每个次要关键词出现3-6次 |
| LSI/语义关键词 | 自然出现 | 不设密度目标，自然写作 |

### 7.3 LSI（语义相关）关键词列表

以下关键词应在文章中自然出现，帮助搜索引擎理解内容主题：

NaHCO₃, baking soda, bicarbonate of soda, sodium hydrogen carbonate,

soda ash, Na₂CO₃, trona, nahcolite, Solvay process,

### 7.4 关键词蚕食（Cannibalization）防范

| 策略 | 说明 |
|------|------|
| 每篇文章有唯一的主关键词 | 不同文章不竞争同一主关键词 |
| 搜索意图区分 | 即使关键词相似，搜索意图不同则可分别优化 |
| 内链引导 | 通过内链告诉Google哪篇是该主题的权威页面 |
| 定期检查 | 用 Google Search Console 监控是否有蚕食现象 |

1. **语言**：全部使用英文撰写（面向国际采购商）
2. **语气**：专业、权威、友好，避免过度营销语气
3. **字数**：严格按照每篇文章的字数要求
4. **原创性**：100%原创内容，不得抄袭
5. **关键词植入**：按照第7章的关键词布局策略执行
6. **内链**：按照第6章的内链矩阵，在文章中自然植入内链
7. **SEO元素**：每篇文章必须提供完整的 Title、Meta Description、URL Slug

9. **CTA**：每篇文章至少包含2个CTA（文中 + 文末）

6. 内链标注（标明链接位置和目标URL）
7. Schema JSON-LD 代码

```
