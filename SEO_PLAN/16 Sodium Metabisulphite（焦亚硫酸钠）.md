## 三、SEO 顶级设置规范

### 3.1 On-Page SEO 标准（每篇文章必须遵守）

#### Title Tag（标题标签）
- 格式：`Primary Keyword - Secondary Keyword | Brand Name`
- 长度：50-60 字符
- 示例：`Sodium Metabisulphite for Water Treatment - Dosage Guide | [公司名]`

#### Meta Description（元描述）
- 长度：150-160 字符
- 包含核心关键词 + 行动号召（CTA）
- 示例：`Learn how sodium metabisulphite is used in water treatment for dechlorination & RO membrane protection. Free dosage guide. Request a quote today.`

#### URL Slug（URL 路径）
- 简洁、包含关键词、用连字符分隔
- 示例：`/blog/sodium-metabisulphite-water-treatment-guide`

#### Heading 结构
```
H1: 文章标题（包含核心关键词，每页仅一个）
  H2: 主要章节标题（包含次要关键词）
    H3: 子章节标题（包含长尾关键词）
      H4: 细分内容（如需要）
```

#### 关键词密度
- 核心关键词：1.5%-2.5%（自然分布）
- 次要关键词：0.5%-1%
- LSI 关键词（语义相关词）：自然穿插
- **首段前100字必须出现核心关键词**
- **H2/H3 标题中自然融入关键词**

#### 图片 SEO
- 每篇文章至少 3-5 张高质量图片
- Alt Text 包含关键词：`alt="food grade sodium metabisulphite powder in 25kg bags"`
- 文件名包含关键词：`sodium-metabisulphite-food-grade-powder.webp`
- 使用 WebP 格式，压缩至 100KB 以下
- 添加图片标题（Image Title）

#### Schema Markup（结构化数据）
- **Article Schema**：每篇 Blog 必加
- **FAQ Schema**：含 FAQ 内容的文章必加（可获得 Google 富摘要）
- **Product Schema**：产品相关页面
- **HowTo Schema**：教程类文章
- **Table Schema**：含对比表格的文章
- **Breadcrumb Schema**：面包屑导航

## 五、关键词布局策略

### 5.1 关键词分配原则

每篇文章分配：
- **1 个核心关键词**（Primary Keyword）
- **2-3 个次要关键词**（Secondary Keywords）
- **5-10 个 LSI / 语义相关词**（Latent Semantic Indexing）
- **3-5 个长尾问题关键词**（Question Keywords）

### 5.2 关键词在文章中的布局位置

```
┌─────────────────────────────────────────────────┐
│  Title Tag        → 核心关键词（靠前位置）         │
│  Meta Description → 核心关键词 + CTA              │
│  URL Slug         → 核心关键词（简化版）           │
│  H1               → 核心关键词（自然融入）         │

│  H2 标题          → 次要关键词 / 长尾关键词        │
│  H3 标题          → 长尾关键词 / 问题关键词        │
│  正文段落          → LSI 词自然分布                │
│  图片 Alt Text    → 核心/次要关键词变体            │
│  内链锚文本       → 关键词变体                     │
```

## 十一、拼写变体覆盖策略

| 变体 | 使用场景 |
|------|---------|
