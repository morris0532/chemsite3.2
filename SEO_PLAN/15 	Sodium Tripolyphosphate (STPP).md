## 1. SEO整体策略概述

### 1.1 策略目标

- **短期（1-3个月）：** 覆盖核心产品词和高转化采购意图词，建立基础内容矩阵
- **中期（3-6个月）：** 深耕行业应用场景词，建立专业权威性（Topical Authority）
- **长期（6-12个月）：** 全面覆盖长尾关键词，形成内容护城河，获取稳定自然流量

### 1.2 内容策略核心原则

- **Pillar-Cluster模型：** 以核心支柱页（Pillar Page）为中心，围绕其构建多个集群内容（Cluster Content），通过内链紧密连接
- **搜索意图匹配：** 每篇Blog精准匹配用户搜索意图（信息型/商业调查型/交易型）
- **E-E-A-T原则：** 体现Experience（经验）、Expertise（专业）、Authoritativeness（权威）、Trustworthiness（可信）
- **全球化视角：** 内容面向全球采购商，覆盖不同地区的采购习惯和法规要求

### 1.3 目标用户画像

## 6. 页面SEO设置规范

### 6.1 Title Tag 规范

- **长度：** 50-60字符（包含空格）
- **格式：** `主要关键词: 补充描述 | 品牌名` 或 `主要关键词: 补充描述 [年份]`
- **要求：**
  - 主要关键词尽量靠前
  - 包含吸引点击的修饰词（Complete, Ultimate, Guide, How to, vs, [年份]）
  - 每篇Blog的Title Tag必须唯一
  - 避免关键词堆砌

### 6.2 Meta Description 规范

- **长度：** 150-160字符
- **格式：** 包含主要关键词 + 价值主张 + CTA暗示
- **要求：**
  - 自然包含1-2个目标关键词
  - 包含具体数据或数字（提高CTR）
  - 以动词开头（Discover, Learn, Compare, Get, Find）
  - 包含品牌差异化信息（如"from a trusted China manufacturer"）

### 6.3 URL Slug 规范

- **格式：** `/blog/关键词-短语`
- **要求：**
  - 全小写，用连字符(-)分隔
  - 包含主要关键词
  - 简短明了（3-6个词）
  - 避免停用词（the, a, an, in, of等可省略）
  - 不包含日期（方便内容更新）

**URL示例：**

| Blog | URL Slug |
|------|----------|
| P1 | /blog/sodium-tripolyphosphate-stpp-guide |
| A1 | /blog/stpp-in-detergent-builder |
| A3 | /blog/stpp-seafood-processing-moisture-retention |
| B1 | /blog/choose-reliable-stpp-supplier-china |
| B2 | /blog/stpp-price-guide |
| C2 | /blog/stpp-phase-1-vs-phase-2 |
| D1 | /blog/stpp-vs-shmp-comparison |
| F1 | /blog/stpp-halal-certification |

### 6.4 H标签层级规范

```
H1: 页面主标题（每页仅1个，包含主要关键词）
  H2: 主要章节标题（包含次要关键词，3-8个）
    H3: 子章节标题（包含长尾关键词，按需使用）
      H4: 细分内容标题（极少使用，仅在内容层级需要时）
```

### 6.5 图片SEO规范

- **文件名：** 使用描述性关键词命名（如 `stpp-detergent-builder-function.jpg`）
- **Alt Text：** 描述图片内容 + 自然包含关键词（如 "Diagram showing how STPP works as a detergent builder to soften water"）
- **图片大小：** 压缩至100KB以下，使用WebP格式
- **每篇文章：** 至少包含2-4张相关图片（流程图、对比表、产品图、应用场景图）

### 6.6 Schema Markup（结构化数据）

每篇Blog建议添加以下Schema：

1. **Article Schema：** 标记文章类型、作者、发布日期、修改日期

3. **Breadcrumb Schema：** 面包屑导航，帮助搜索引擎理解网站结构
4. **Product Schema：** 产品相关页面，标记产品名称、描述、品牌
5. **Organization Schema：** 全站标记公司信息
