# Что такое Каустическая Сода? Полное Руководство для Промышленных Покупателей

**Каустическая сода**, химически известная как **гидроксид натрия (NaOH)**, является одним из наиболее важных промышленных химикатов в мире. От водоочистных сооружений до производства литиевых батарей, каустическая сода играет незаменимую роль в современной промышленности. Это всеобъемлющее руководство подробно описывает **что такое каустическая сода** — ее свойства, применение, марки и способы эффективного приобретения для вашего бизнеса. Эта статья является вашим основным источником для полного понимания гидроксида натрия.

## Что такое Каустическая Сода? Химическое Определение и Основные Свойства

Каустическая сода — это неорганическое соединение с химической формулой NaOH. Это сильное щелочное вещество, которое существует в нескольких формах: твердые чешуйки, гранулы или жидкие растворы. Термин "каустическая" относится к ее коррозионной природе, поэтому обращение и хранение требуют особого внимания.

### Ключевые Химические Свойства

| Свойство | Значение |
| :--- | :--- |
| **Химическая Формула** | NaOH |
| **Молекулярный Вес** | 40 г/моль |
| **Номер CAS** | 1310-73-2 |
| **Внешний Вид** | Белое твердое вещество или бесцветная жидкость |
| **Растворимость** | Чрезвычайно растворима в воде (экзотермическая реакция) |
| **Плотность (жидкость)** | 1.2 г/см³ |
| **Плотность (твердое)** | 2.13 г/см³ |
| **Температура Плавления** | 318°C |
| **Температура Кипения** | 1,388°C |
| **pH (1% раствор)** | 13.0 |

При растворении в воде каустическая сода выделяет значительное количество тепла — реакция, известная как экзотермический процесс. Эта характеристика делает необходимым добавлять каустическую соду в воду, а не наоборот, чтобы предотвратить опасное разбрызгивание.
