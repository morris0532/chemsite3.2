Как подробно описано в нашем **[Полном руководстве по кальцинированной соде (карбонату натрия) для B2B-покупателей](/ru/blog/the-ultimate-soda-ash-sodium-carbonate-guide-for-b2b-buyers-2025-2026-strategic-sourcing-manual)**, в мировой химической торговле Сертификат анализа (COA) и Технический паспорт (TDS) являются единственными юридическими гарантиями качества продукции. Для закупок [кальцинированной соды](/ru/products/soda-ash-sodium-carbonate) (карбоната натрия) понимание следовых примесей, таких как железо (Fe), хлориды (Cl) и сульфаты (SO4), имеет решающее значение для предотвращения производственных сбоев. Это руководство разбивает основные параметры, которые каждый B2B-покупатель должен проверить, прежде чем подписывать оптовый контракт.

## Основные спецификации кальцинированной соды: COA против TDS

| Параметр спецификации | Стандартный промышленный сорт | Высокочистый / Солнечный сорт | Влияние на производство |
| :--- | :--- | :--- | :--- |
| **Общая щелочность (Na2CO3)** | ≥ 99.2% | ≥ 99.5% | Концентрация основного активного ингредиента |
| **Содержание железа (Fe)** | ≤ 0.003% (30ppm) | ≤ 0.0015% (15ppm) | Критично для прозрачности и цвета стекла |
| **Хлориды (NaCl)** | ≤ 0.7% | ≤ 0.2% | Коррозионная активность в химических реакторах |
| **Сульфаты (SO4)** | ≤ 0.03% | ≤ 0.01% | Влияет на чистоту [химического синтеза](/ru/blog/soda-ash-in-chemical-synthesis) |
| **Нерастворимые в воде вещества** | ≤ 0.03% | ≤ 0.01% | Предотвращает засорение в системах фильтрации |
| **Насыпная плотность (Тяжелая)** | 1.0 - 1.2 г/см³ | 1.05 - 1.15 г/см³ | Эффективность плавления в печи и хранения |

## 1. TDS против COA: В чем разница для покупателя?
Прежде чем размещать заказ у **китайского поставщика кальцинированной соды**, вы должны понять различие между этими двумя документами.

*   **Технический паспорт (TDS)**: Общий лист спецификаций, предоставляемый производителем. Он описывает *стандартный* диапазон продукта. Это маркетинговый и технический справочник.
*   **Сертификат анализа (COA)**: Документ, специфичный для партии. Он показывает *фактические* результаты испытаний конкретной партии кальцинированной соды, которую вы покупаете.

### Стратегия закупок:
Всегда запрашивайте **COA, специфичный для партии**, прежде чем груз покинет **порт Тяньцзинь**. Общего TDS недостаточно для обеспечения качества для ответственного производства.

## 2. Расшифровка ключевых химических параметров
«Большая тройка» параметров определяет ценность и применение вашего карбоната натрия.

### А. Общая щелочность (как Na2CO3)
Это чистота продукта. Стандартный промышленный сорт обычно составляет **минимум 99.2%**.
*   **Проблема**: Низкая чистота означает, что вы платите за «наполнители» и примеси, которые могут изменить стехиометрию ваших химических реакций.
*   **Решение**: Убедитесь, что COA показывает стабильный уровень чистоты выше 99.2% для поддержания стабильности процесса.

### Б. Содержание железа (Fe) – Приоритет для производителей стекла
Железо — самая опасная примесь в производстве стекла.
*   **Влияние**: Даже 0.005% Fe могут окрасить прозрачное стекло в зеленый цвет. Для **солнечного стекла** требования еще строже (≤ 0.0015%).
*   **B2B намерение**: Менеджеры по закупкам для стекольных заводов должны отдавать приоритет поставщикам, использующим **процесс Хоу** или высококачественную природную трону, так как они обычно дают более низкое содержание железа.

### В. Хлориды (как NaCl)
Хлориды являются побочным продуктом процесса Сольве.
*   **Влияние**: Высокое содержание хлоридов вызывает коррозионное растрескивание оборудования из нержавеющей стали под напряжением.
*   **Цель**: Для химического синтеза и металлургии стремитесь к содержанию хлоридов ниже 0.5% в COA.

## 3. Физические характеристики: Плотность и размер частиц
Химическая чистота — это только половина дела. Физическая «форма» кальцинированной соды определяет, как она ведет себя на вашем заводе.

### А. Насыпная плотность (Тяжелая против Легкой)
*   **Тяжелая**: Целевое значение 1.0 - 1.2 г/см³. Если COA показывает плотность ниже 0.9, это может вызвать «пыление» в вашей печи.
*   **Легкая**: Целевое значение 0.5 - 0.6 г/см³. Необходима для быстрого растворения при смешивании моющих средств.

### Б. Ситовой анализ (Распределение частиц по размеру)
Профессиональный COA будет включать ситовой тест.
*   **Почему это важно**: Для производства стекла размер частиц тяжелой кальцинированной соды должен соответствовать кварцевому песку (обычно 300-1000 микрон), чтобы предотвратить расслоение сырья.

## 4. Как распознать «поддельный» или некачественный COA
На рынке **оптовых закупок кальцинированной соды** некоторые поставщики могут предоставлять «шаблонные» COA, которые не отражают фактический груз.

### Красные флаги, на которые следует обратить внимание:
1.  **Идентичные результаты**: Если результаты для Na2CO3, Fe и NaCl абсолютно одинаковы в нескольких партиях (например, всегда ровно 99.20%, 0.0030%), COA, скорее всего, является шаблоном.
2.  **Отсутствие номеров партий**: Каждый COA должен быть привязан к конкретной производственной партии и дате упаковки.
3.  **Нет печати лаборатории**: Действительный COA должен быть подписан менеджером по контролю качества (QC) и заверен печатью заводской лаборатории.

## 5. Стандарты соответствия: ISO, REACH и SGS
Ведущий **китайский завод по производству кальцинированной соды** подтверждает свой COA международными сертификатами.

*   **ISO 9001:2015**: Гарантирует, что завод имеет последовательную систему управления качеством.
*   **Регистрация REACH**: Обязательна для любого покупателя в Европейском Союзе. В COA должен быть указан регистрационный номер REACH.
*   **Инспекция третьей стороной (SGS/BV)**: Для новых B2B-отношений мы рекомендуем нанимать SGS для проведения независимого отбора проб и анализа в порту погрузки.
