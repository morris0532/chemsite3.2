## 四、通用写作规范

### 4.1 On-Page SEO 规范

| 项目 | 要求 |
|------|------|
| **Title Tag** | 包含主关键词，55-60 字符以内 |
| **Meta Description** | 包含主关键词 + CTA 引导语，150-160 字符 |
| **H1 标签** | 每页仅 1 个 H1，必须包含主关键词 |
| **H2/H3 标签** | 合理使用层级结构，H2/H3 中自然嵌入 LSI 关键词 |
| **关键词密度** | 主关键词 1-2%，均匀分布在全文中 |
| **首段规则** | 前 100 字内必须出现主关键词 |
| **图片** | 每篇 2-4 张图片，每张图片必须有描述性 alt 标签（含关键词） |
| **内链** | 每篇 4-6 条内链，锚文本自然融入正文 |
| **外链** | 每篇 1-2 条权威外链（Wikipedia、行业协会、政府机构等） |
| **URL 格式** | `yourdomain.com/blog/[primary-keyword-slug]`，全小写，用连字符分隔 |
| **Schema** | 添加 Article schema；如有 FAQ 段落，添加 FAQ schema |

### 4.2 内容质量标准

| 项目 | 要求 |
|------|------|
| **原创性** | 100% 原创，禁止抄袭或洗稿 |
| **专业性** | 数据准确、化学术语正确、引用可靠来源 |
| **可读性** | 短段落（3-5 句）、善用列表和表格、Flesch Reading Ease > 50 |
| **语言** | 英文，面向国际 B2B 买家，专业但易懂 |
| **格式** | 必须使用 H1/H2/H3 层级、粗体强调关键信息、表格呈现数据 |

### 4.3 标题规范（重点）

**规则：** 文章标题只在 H1 出现一次，内容中不重复

**错误示例：**
```
<h1>Caustic Soda Applications: Complete Industry Guide</h1>
<p>Caustic Soda Applications are used in many industries...</p>  ❌ 重复了标题
<h2>Caustic Soda Applications in Water Treatment</h2>  ❌ 重复了关键词
```

**正确示例：**
```
<h1>Caustic Soda Applications: Complete Industry Guide</h1>
<p>Sodium hydroxide serves critical roles across multiple sectors...</p>  ✅ 同义词替换
<h2>Water Treatment & pH Control</h2>  ✅ 使用不同表述
<p>Industrial water treatment facilities rely on alkaline solutions...</p>
```
