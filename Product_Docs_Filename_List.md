# Sinopeakchem 产品技术文档命名清单

**注意：**
1. 所有文件名必须使用**小写字母**。
2. 单词之间使用**连字符 `-`** 连接。
3. 每个产品对应两个文件：`[slug]-msds.pdf` 和 `[slug]-coa.pdf`。

### 操作步骤：
1. 准备好您的英文版 PDF 文档。
2. 将文件重命名为上表中的名称。
3. 将文件放入项目的 `public/docs/` 目录下。
4. 提交并推送至 GitHub，Vercel 部署后功能即可生效。
